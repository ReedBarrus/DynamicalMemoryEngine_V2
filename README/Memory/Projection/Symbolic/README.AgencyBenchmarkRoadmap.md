2. **Helper posture packet** — restates role law, packet workflow, and downgrade posture for any human or model helper.
3. **Benchmark data governance** — proves how benchmark datasets will stay provenance-aligned and below canon during execution.
4. **Evaluation alignment** — binds the revived benchmarks to the current evaluation surfaces so they do not become shadow review.

Until all four gates are satisfied, this survivor remains a roadmap summary and the archived notes remain the substantive references.
