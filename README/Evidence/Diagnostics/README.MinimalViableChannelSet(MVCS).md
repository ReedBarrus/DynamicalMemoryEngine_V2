## 3. Core rules

    ### Rule 1 — Channel necessity rule

    A channel should remain in MVCS only if it preserves an identity axis that another current channel does not already preserve.

    ### Rule 2 — Attribution rule

    Channels must remain explicitly attributable.
    A result should remain traceable to the channel that preserved it.

    ### Rule 3 — No premature fusion

    Channels must not be fused into one composite representation during the current identity-stabilization phase.

    ### Rule 4 — Identity is not measurement

    A change in measured expression does not automatically imply identity collapse.
    A system may appear unstable purely because its measurement space is inconsistent.

    ### Rule 5 — Scale is observation geometry

    Scale changes how identity becomes observable.
    Scale does not automatically create a new identity channel.

## 8. Spectral calibration constraint

    Uncalibrated spectral distances must not be used as if they were globally portable across scales.

    All cross-scale spectral comparison must use one of:

    ### A. Per-scale thresholds

    Each declared scale uses its own explicit comparison threshold.

    ### B. Scale-normalized spectral distance

    A calibration such as bin-width normalization may be used so that spectral distance becomes comparable across observation horizons.

    Current preferred physically grounded posture:

    * normalize by bin width (`Fs / N`) in probes, (This normalization converts spectral distance from a representation-dependent unit (per-bin) into a physically grounded unit (per-Hz), removing dependence on window resolution.)
    * emit both raw and normalized values,
    * preserve bin width explicitly in receipts.

    Relative-within-scale ranking alone is not sufficient for identity-boundary classification when the question is absolute separability rather than within-scale ordering.

## 8.5 Probe note — calibrated spectral comparison and support-horizon resonance

    Recent probe work establishes two important clarifications for the current MVCS phase.

    ### A. Spectral identity did not collapse

    Earlier apparent cross-scale inconsistency in spectral separability was not a failure of identity.
    It was a failure of comparison posture.

    After calibrated comparison (especially bin-width normalization), the spectral channel remains separable across the declared scales for the relevant frequency-bearing pairs.
    This means the underlying frequency / shape identity is stable, while its raw measurement expression is scale-conditioned.

    Therefore:

    - raw spectral distance must not be treated as globally portable across scale,
    - calibrated spectral comparison is now part of the required probe posture,
    - apparent fragmentation at the between-cohort level must first be tested against calibration before being interpreted as real structural instability.

    ### B. Some basin fragmentation is lawful

    Once spectral comparison is calibrated, one class of fragmentation remains real:

    - within-cohort basin splitting at a specific support horizon.

    This means not all apparent instability is threshold artifact.
    Some fragmentation reflects genuine structure emerging under a particular observation horizon.

    Current evidence supports the following distinction:

    1. **threshold artifact**
    - instability disappears after calibrated comparison

    2. **support-horizon resonance**
    - instability persists after calibration and appears only at specific scale / period relationships

    3. **unresolved instability**
    - instability remains but is not yet explained by calibration or support-horizon structure

    ### C. Support-horizon resonance is now a lawful explanatory category

    Current probe evidence supports the working interpretation that basin splitting may emerge when:

    - the declared window duration approaches the dominant recurrence period,
    - and the cohort’s harmonic arrangement permits phase-conditioned variation to remain visible across windows.

    This means:

    - phase-ratio alignment may be necessary for one class of splitting,
    - but phase-ratio alignment alone is not sufficient,
    - harmonic placement and internal spectral arrangement also matter.

    ### D. Practical rule for current MVCS work

    For the current phase:

    - do not add new channels,
    - do not add derivatives,
    - do not add phase as a separate identity channel,
    - do not fuse the frozen base channels,
    - first classify any apparent instability as:
    - threshold artifact,
    - support-horizon resonance,
    - lawful fragmentation / consolidation,
    - or unresolved.

    This keeps identity work attributable and prevents the system from solving a measurement inconsistency by inflating basis complexity.

    ### E. Updated working principle

    Identity in the current Door One probe phase is stable and well formed.
    What varies is:

    - how identity is measured across scale,
    - and whether a given support horizon exposes lawful internal fragmentation.

    That distinction is now part of the active MVCS probe methodology.

## Probe note — boundary-conditioned splitting and harmonic placement

    Recent probe work sharpens the current MVCS posture in an important way.

    ### A. Spectral instability was not identity collapse

    Calibrated spectral comparison showed that earlier cross-scale inconsistency in spectral separability was a comparison artifact, not a true failure of identity.
    The current two-channel base remains sufficient:
    - absolute energy preserves amplitude-bearing identity,
    - spectral distribution preserves frequency / shape identity when compared lawfully across scale.

    ### B. Some remaining basin splitting is real

    After calibration, one class of basin fragmentation remained:
    - within-cohort splitting under a specific support horizon.

    This means the current instability picture must now be split into:
    1. threshold artifact,
    2. lawful support-horizon fragmentation,
    3. unresolved cause.

    ### C. Harmonic-placement probing refined the splitting rule

    The latest probe shows that splitting is not a broad “near-boundary” effect.

    Instead, for the current setup, splitting appears as a sharp edge-conditioned event:
    - nearby harmonic placements produce elevated stress,
    - but lawful splitting occurs only at the exact edge case.

    This means the current structural explanation must distinguish between:
    - near-boundary stress,
    - exact-edge bifurcation,
    - and consolidation away from the edge.

    ### D. Exact edge placement is still not sufficient by itself

    The harmonic-placement probe also showed:
    - not every formal edge is active,
    - not every exact edge splits,
    - and amplitude modulates the response within a finite window.

    This refines the current working rule:

    Splitting currently appears to require:
    - support-horizon resonance,
    - exact boundary placement in the active band-profile space,
    - a finite amplitude window,
    - and an interaction topology that permits time-conditioned redistribution across the boundary.

    ### E. Dominant-band switching is not the mechanism

    The current evidence indicates that lawful splitting is not explained by windows flipping dominant bands.

    Instead, splitting is better described as:
    - temporally bimodal band-profile distributions across windows,
    - produced by boundary-conditioned redistribution in the profile geometry.

    This matters because it means basin formation is responding to full profile organization over time, not only to peak-band changes.

    ### F. Updated MVCS implication

    The current MVCS freeze remains correct.

    Do not:
    - add derivatives,
    - add phase as a separate channel,
    - add embeddings,
    - fuse channels.

    Instead:
    - preserve channel separation,
    - preserve calibrated spectral comparison,
    - and treat boundary-conditioned basin behavior as the current explanatory frontier.

    ### G. Active next question

    The next probe question is no longer whether edge placement matters.
    That has already been established.

    The next question is:

    **Which exact boundaries are active interaction zones, and which are inert?**

    That distinction is now the next lawful step in understanding basin formation as energy topology over time.
