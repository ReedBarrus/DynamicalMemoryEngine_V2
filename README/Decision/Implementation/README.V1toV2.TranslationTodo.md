Translate V1 into V2 by:

auditing V1 seam by seam
classifying what survives
extracting only lawful substrate
rewriting surviving laws in smaller V2 form
attaching explicit tests
exposing only legible projections
refusing silent authority inflation

The translation target is not “port V1.”

The translation target is:

preserve surviving law, discard confusion, and compress architecture into auditable V2 surfaces.

## 3. Core translation rule

Every V1 seam under review must answer:

what was admitted?
what transformed?
what object was emitted?
what class was that object?
what remained invariant?
what provenance was preserved?
what memory posture was reached?
what did the seam not claim?

If these cannot be answered clearly, the seam is not ready for translation.

## 4. Object class rule

Every audited V1 artifact must be classified as exactly one of:

Primary object

Direct active structure in the operator chain.

Derived support object

Secondary structural object used for comparison, compression, diagnostics, feature support, exposure, or evaluation support.

Interpretive object

Overlay, narrative, symbolic, semantic, or review-layer object.

Class audit question

Did this object act above its class?

If yes, mark the seam as mixed or unsound.

## 5. Seam classification rule

Each audited seam must be marked:

Intact

The seam preserves class boundaries, provenance, invariant posture, and emitted-object honesty.

Mixed but recoverable

The seam contains lawful substrate, but class boundaries, projection boundaries, or authority posture became mixed.

Structurally unsound

The seam cannot be translated honestly without redesign because identity, provenance, class posture, or authority posture collapses.

## 6. Translation buckets

Every V1 seam or artifact goes into one of three buckets:

Carry forward

Survives audit and should be restated in V2.

Quarantine

Still useful for evidence, comparison, or test pressure, but not ready for port.

Re-derive

The intention was valuable, but the V1 implementation is too mixed or unclear to port directly.

### Phase C — Extract laws, not bulk
C.1 Extract surviving laws
